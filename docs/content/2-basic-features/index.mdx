## 功能架构

<Callout type="info">
  基础功能模块是Mini-Feeds系统的核心部分，为推荐系统、搜索广告和大模型应用等高级功能提供基础支持。
</Callout>

基础功能模块之间的关系如下：

1. **Feeds流**：系统的核心展示层，负责将内容、广告和商品以统一的方式呈现给用户
2. **用户交互系统**：捕获用户的主动行为（如点赞、收藏），并更新用户与内容的关系
3. **用户埋点**：记录用户的被动行为（如曝光、停留时间），为推荐和广告系统提供数据支持
4. **基础指标看板**：基于埋点数据构建业务指标，监控系统运行状况

在接下来的章节中，我们将详细介绍每个功能模块的设计思路、实现方式和关键代码。
